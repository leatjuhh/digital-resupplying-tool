# Database Setup - SQLite + SQLAlchemy

## 🎯 Wat Gaan We Bouwen

Van dummy data naar echte database met:
- SQLite database (geen extra installatie nodig)
- SQLAlchemy ORM voor type-safe database operaties
- Stores tabel voor winkellocaties
- Articles tabel voor artikelen met voorraad
- CRUD endpoints (Create, Read, Update, Delete)
